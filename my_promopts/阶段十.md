4. 编写 Bug 修复说明
新建 docs/bug-fixes.md，覆盖 attachments/buggy-component.tsx 中 3 个 bug。每个 bug 按以下结构写：
- 问题现象
- 根因分析
- 修复方法
- 当前项目中的对应实现位置

必须包含：
1. fetchAlerts 缺少 buildingId 依赖
   - 原问题：楼栋变化后仍请求旧楼栋告警。
   - 修复：当前 DeviceAlertPanel 使用 queryKey ['alerts', buildingId] 和 getAlerts({ buildingId })，buildingId 变化自动重新请求。

2. 自动刷新定时器没有清理
   - 原问题：setInterval 未 clearInterval，组件卸载或依赖变化后仍刷新。
   - 修复：当前使用 React Query refetchInterval: autoRefresh ? 5000 : false，关闭自动刷新或组件卸载时由 React Query 管理停止刷新。

3. 确认告警时直接修改 React state
   - 原问题：alert.acknowledged = true 后 setAlerts(alerts)，直接 mutation，可能不触发正确渲染。
   - 修复：当前 ack mutation 成功后用 queryClient.setQueryData 和 prev.map 返回新对象，不可变更新。
