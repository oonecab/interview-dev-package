### AI Tool Calling

- [x] 发送"B3栋有哪些故障设备？"→ 工具卡片显示"查询设备"→ 显示设备列表回复
- [x] 发送"B1栋有哪些告警？"→ 工具卡片显示"查询告警"→ 显示告警列表回复
- [x] 发送"给电梯_001创建一个高优先级工单"→ 工具卡片显示"创建工单"→ 显示创建成功
- [x] 发送"对B1栋出现故障的设备创建工单"→ 当 LLM 返回空 `deviceId` 时，先查询故障设备，再使用真实 `deviceId` 创建工单
- [x] 发送"对B2栋出现故障的设备创建工单"且没有匹配设备 → 显示“没有匹配设备，未创建工单”，不展示失败兜底，不调用创建工单接口
- [x] AI 工具执行器覆盖 `api-spec.md` 全部业务接口：楼栋、设备列表、设备详情、告警查询、告警确认、工单查询、工单创建、工单状态更新
- [x] 工具调用失败时显示失败状态，不会继续假成功
- [x] create_work_order 成功后工单列表自动刷新
